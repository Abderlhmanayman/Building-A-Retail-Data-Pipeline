# Building a Retail Data Pipeline (Walmart Data)

## 📝 Project Overview
Building an end-to-end **ETL pipeline** for retail data, integrating multiple source formats to enable data-driven insights.

## 🎯 Project Milestones

### 1️⃣ Data Collection & Preprocessing
- Ingesting data from **SQL** and **Parquet** files.
- Cleaning and unifying data formats for consistency.

### 2️⃣ System Development & Evaluation
- Transforming raw data into structured formats ready for analytics.

### 3️⃣ Deployment
- Loading the processed data into an **Analytics Datastore (Warehouse)**.

### 4️⃣ MLOps & Automation
- Setting up an automated pipeline using **Apache Airflow**.

### 5️⃣ Final Documentation & Demo
- Presenting the pipeline structure and final outputs.

## 🛠️ Tech Stack
- **Languages:** Python, SQL
- **Orchestration:** Apache Airflow
- **Storage:** Data Warehouse
- **Formats:** Parquet, SQL
